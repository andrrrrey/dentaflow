AI-Робот — Демо голосового контура

Этап 1: Проверка TTS, ASR, сценариев
Проверка...

1 Синтез речи (TTS)

Введите текст — робот озвучит его голосом. Проверьте темп, интонацию, естественность.

2 Распознавание речи (ASR)

Запишите голос с микрофона — робот распознает речь. Или используйте аудио из TTS-теста.
Запись... нажмите «Остановить» для распознавания

3 Полный цикл: TTS → ASR

Робот озвучивает фразу, затем распознаёт собственную речь. Результат должен совпадать с оригиналом.
Текст → TTS → Аудио → ASR → Текст

4 Синтез речи — SaluteSpeech (Sber)

Голоса Sber SmartSpeech API. Требуется SALUTESPEECH_AUTH_KEY в окружении.

5 Сценарии диалогов

Загруженные сценарии обзвона. Заказчик проверяет и согласовывает.
